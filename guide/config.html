Configuration ​

Vince acn be configured with commandline flags or environment variables.

TIP

We recommend the use of environment variables. They are safer (passing secrets ), and it allows simpler deployments, where you set the environment and just call vince command.

Alerts ​

alerts-source ​

path to directory with alerts scripts

shell
--alerts-source="..."
shell
VINCE_ALERTS_SOURCE="..."

enable-alerts ​

allows loading and executing alerts

shell
--enable-alerts="false"
shell
VINCE_ENABLE_ALERTS="false"

Backup ​

backup-dir ​

directory where backups are stored

shell
--backup-dir="..."
shell
VINCE_BACKUP_DIR="..."

enable-backup ​

Allows backing up and restoring

shell
--enable-backup="false"
shell
VINCE_BACKUP_ENABLED="false"

Bootstrap ​

bootstrap-email ​

Email address of the user to bootstrap.

shell
--bootstrap-email="..."
shell
VINCE_BOOTSTRAP_EMAIL="..."

bootstrap-key ​

API Key of the user to bootstrap.

shell
--bootstrap-key="..."
shell
VINCE_BOOTSTRAP_KEY="..."

bootstrap-name ​

Full name of the user to bootstrap.

shell
--bootstrap-name="..."
shell
VINCE_BOOTSTRAP_NAME="..."

bootstrap-password ​

Password of the user to bootstrap.

shell
--bootstrap-password="..."
shell
VINCE_BOOTSTRAP_PASSWORD="..."

enable-bootstrap ​

allows creating a user and api key on startup.

shell
--enable-bootstrap="true"
shell
VINCE_ENABLE_BOOTSTRAP="true"

Core ​

data ​

path to data directory

shell
--data=".vince"
shell
VINCE_DATA=".vince"

enable-profile ​

Expose /debug/pprof endpoint

shell
--enable-profile="false"
shell
VINCE_ENABLE_PROFILE="false"

listen ​

http address to listen to

shell
--listen=":8080"
shell
VINCE_LISTEN=":8080"

log-level ​

log level, values are (trace,debug,info,warn,error,fatal,panic)

shell
--log-level="debug"
shell
VINCE_LOG_LEVEL="debug"

super-users ​

a list of user ID with super privilege

shell
--super-users="..."
shell
VINCE_SUPER_USERS="..."

url ​

url for the server on which vince is hosted(it shows up on emails)

shell
--url="http://localhost:8080"
shell
VINCE_URL="http://localhost:8080"
